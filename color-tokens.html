📝 사용 가이드

이 색상 토큰들을 사용하여 일관된 디자인 시스템을 유지하세요:

  • Primary Gradient: 주요 액션 버튼, 브랜드 요소
  • Gray Scale: 텍스트, 배경, 테두리
  • Light Green: 정보 카드, 설명 섹션
  • Yellow: 경고, 테스트넷 배지
  • Fonts: 언어별 적절한 폰트 사용

🎯 주요 변경 포인트

색상 테마를 변경하려면 다음 파일들을 수정하세요:

  • apps/nextjs/src/app/globals.css - CSS 변수 정의
  • apps/nextjs/tailwind.config.ts - Tailwind 설정
  • tooling/tailwind/base.ts - 기본 Tailwind 설정
  • apps/nextjs/src/context/LanguageProvider.tsx - 언어별 폰트 설정

🔧 폰트 변경 방법

폰트를 변경하려면 다음 파일들을 수정하세요:

  • apps/nextjs/src/app/globals.css - Google Fonts import 및 CSS 변수
  • apps/nextjs/tailwind.config.ts - Tailwind 폰트 설정
  • apps/nextjs/src/context/LanguageProvider.tsx - 언어별 폰트 설정
