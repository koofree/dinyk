📝 사용 가이드

이 색상 토큰들을 사용하여 일관된 디자인 시스템을 유지하세요:

  • Primary: 주요 액션 버튼, 브랜드 요소
  • Secondary: 보조 액션, 카드 배경
  • Success: 성공 메시지, 긍정적 수치
  • Warning: 경고 메시지, 주의사항
  • Error: 에러 메시지, 삭제 액션
  • Gray Scale: 텍스트, 배경, 테두리

🎯 주요 변경 포인트

색상 테마를 변경하려면 다음 파일들을 수정하세요:

  • apps/nextjs/src/app/globals.css - CSS 변수 정의
  • apps/expo/src/styles.css - Expo 앱 CSS 변수
  • tooling/tailwind/base.ts - Tailwind 기본 설정
  • apps/nextjs/src/lib/constants.ts - 리스크 레벨 색상
